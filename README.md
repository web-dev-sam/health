# Health Wiki

A personal wiki for health-related topics — nutrition, supplementation, and evidence-informed lifestyle decisions.

Built with Vue 3 + TypeScript + Tailwind CSS, with bilingual support (EN/DE).

## Topics

- **Sweeteners** — ranked guide to natural and artificial sweeteners with consumption guidelines

## Development

```sh
bun install
```

```sh
bun dev
```

```sh
bun run build
```
